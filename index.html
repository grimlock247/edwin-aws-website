About the Brand

Edwin Esports Merch is a mock esports merchandise storefront created as part of an AWS cloud computing assignment. The goal of the project is to demonstrate how a static website can be built and hosted using Amazon Web Services while also reflecting a personal interest in esports and gaming culture.
